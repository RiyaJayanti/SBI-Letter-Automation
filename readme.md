# SBI Letter Automation System

A comprehensive banking letter automation solution for State Bank of India branch management.

## Features
- Excel customer data processing
- Intelligent issue detection
- Automated letter generation
- Bulk email sending
- AI-powered customer analysis
- Interactive chatbot assistance

## Installation & Setup

### 1. Clone the repository
```
git clone https://github.com/YOUR_USERNAME/SBI-Letter-Automation.git
```

### 2. Navigate to project folder
```
cd SBI-Letter-Automation
```

### 3. Install dependencies
```
npm install
```

### 4. Run the project
```
npm start

##  Use Case

- Banking document automation
- Reducing manual letter creation
